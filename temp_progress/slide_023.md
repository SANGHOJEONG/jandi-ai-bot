## 2. 물류/배송: 프리미엄 배송 운영 프로세스

**문서 정보**
*   **대상:** 파트너사
*   **작성자:** 강민호
*   **검수자:** 강민호
*   **수정일:** 26.06.10

---

### 1. 프리미엄 배송 개요
*   **대상:** 300만 원 이상 초고가 상품 안전 대면 배송

| 구분 | 세부 내용 | 비고 |
| :--- | :--- | :--- |
| **서비스 내용** | 300만 원 이상 초고가 상품 대면 배송, 오전 10시 이전 주문 시 익일 배송 가능 (*평일 운영, 전국 배송(제주도 포함)) | 고객 희망 시 지정일 배송 가능 |
| **신청 방법** | 반영 필요 상품코드 온라인 부문 물류 담당자 전달, 브랜드 추가 시 별도 프리미엄 배송 계정 생성 필요 | - |
| **배송 상품 기준** | 센터 경유 배송(업체 배송 상품 수행 불가), 주문 제작 상품 수행 불가, 20kg 이상 및 세 변의 합 160cm 이상 상품 제외, 파손 위험이 큰 물성 상품 제외 | - |
| **운영 점포** | 본점, 잠실, 인천, 부산, 광복점 | - |
| **배송 안내** | 프리미엄 배송 상품은 상품 내 별도 배송 플래그 노출 | 무료배송 / 프리미엄배송 |

---

### 2. 물류 프로세스
물류 처리 단계는 다음과 같이 순차적으로 진행됩니다.

1.  **주문 확인 (10:00)**
    *   고객 주문 내역 확인
    *   집하장 프리미엄 배송 라벨 출력 후 주문 매장 전달 (11:00 ~ 12:00)
2.  **상품 준비**
    *   매장 프리미엄 배송 상품 박스 패킹
    *   배송기사 픽업 전 작업 완료 필요 (13:00 전)
3.  **상품 픽업 (14:00 ~ 18:00)**
    *   배송기사(발렉스) 매장 방문 및 배송상품 픽업
    *   고객 배송 가능 시간 확인(발렉스)
4.  **고객 전달 (D+1~)**
    *   보안요원 대면 배송
    *   고객 요청 시 비대면 가능
