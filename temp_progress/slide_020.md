## 온라인 상품 관리 > 상품 등록(채널상품등록_상품상세설명)

*   **화면 구성 설명**
    *   **상품상세설명**: 상품의 상세 정보를 입력하는 영역입니다.
    *   **상품 카테고리 선택**: 상품의 카테고리를 선택합니다.
    *   **신사정장(상의) 사이즈 (cm)**: 상의 사이즈 정보를 입력합니다.
        *   **표준 사이즈**: 95, 100 등의 표준 사이즈를 선택합니다.
        *   **브랜드 사이즈**: 브랜드별 사이즈를 입력합니다.
        *   **신장**: 신장을 입력합니다.
        *   **가슴둘레**: 가슴둘레를 입력합니다.
        *   **허리둘레**: 허리둘레를 입력합니다.
        *   **※ 사이즈 조건표 데이터 입력**: 사이즈 조건표 데이터를 입력하는 영역입니다.
    *   **신사정장(하의) 사이즈 (cm)**: 하의 사이즈 정보를 입력합니다.
        *   **표준 사이즈**: 30, 31, 32, 33 등의 표준 사이즈를 선택합니다.
        *   **브랜드 사이즈**: 브랜드별 사이즈를 입력합니다.
        *   **허리둘레**: 허리둘레를 입력합니다.
    *   **사이즈조견표 B 미리보기**: 입력한 사이즈 정보를 미리 보기 합니다.
    *   **사이즈조견표 복사**: 입력된 사이즈 정보를 확인 후 복사하여 버튼을 클릭합니다.
        *   **→ 자동으로 상품상세에 이미지로 입력**: 사이즈 조견표가 자동으로 상품상세에 이미지로 입력됩니다.
    *   **※ 상품 상세는 공통 상품정보 등록 창에서 입력한 이미지 및 텍스트 자동 복사**: 롯데ON/아이몰에 해당하는 추가 이미지 및 텍스트 입력.

*   **안내사항**:
    *   등록할 카테고리를 선택하고 사이즈 입력 후, [미리보기] → [사이즈 조견표 복사] 버튼을 눌러주세요.
    *   기입된 내용이 맞는지 확인하시고, html 소스를 복사하시거나 텍스트로 복사생성하세요.
