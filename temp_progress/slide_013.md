## 2. 물류/배송: 백화점 온라인 배송 형태

* **검수자:** 오승근

### 1. 배송 처리 유형 및 배송 완료 처리 방식
* **센터 경유배송**
    * 매장 / 업체 센터에서 상품을 준비하여 점포에 구축된 집하장을 통해 출고 진행
    * 택배사 배송 완료 정보를 직접 연동 받아 처리
* **업체 직접 배송**
    * 업체에서 상품 준비 후 발송 처리 정보를 입력하고 개별 택배사 활용 출고 진행
    * 배송 추적 솔루션을 활용하여 배송 완료 정보를 연동 받아 처리
    * ※ 출고완료 처리 시 정확한 운송장 번호 입력 필요
    * ※ 설치 배송의 경우 직접 배송완료 처리

### 2. 배송 상품 구분
* **일반상품:** 주문 접수 후 수일내 배송 가능한 상품
* **주문제작상품:** 주문 접수 후 제작하여 배송하는 상품
* **설치상품:** 주문 접수 후 고객과 사전 연락을 통해 별도의 설치 과정이 필요한 상품

### 3. 판매 방식
* **일반판매:** 판매 시작일 후 주문과 동시에 출고지시가 적용되는 상품
* **예약판매:** 판매 시작일 후 예약 일자에 출고지시가 일괄 적용되는 상품

### 4. 주문 상태
* **주문완료:** 결제완료 상태
* **취소완료:** 취소완료 상태
* **출고지시:** 출고지시가 떨어진 상태 (주문라벨 출력 전달)
* **상품준비:** 주문을 확인한 상태 (집하장)
* **출고완료:** 상품이 택배사 인계 혹은 직접 배송을 위해 준비된 상태 (물류센터)
* **배송완료:** 고객에게 인도 완료된 상태
