### 3. 반송장 정보 자동 연동 (업체배송)

* **개요**: 이용하는 택배사 신용코드 정보를 PO에 등록하면, 반송장 정보를 자동으로 연동하고 예정일 이내 집하/출고를 못 하는 경우 지연 일자를 지정하여 고객에게 알림.
* **약정일 지정 방법**
    1. PO 배송일정요청 화면에서 약정일 지정이 필요한 주문 클릭
    2. '약정/복구요청' 버튼 클릭하여 날짜 지정 후 '확인' 버튼 클릭
    3. 담당 MD 승인 이후 처리 (※아이몰 주문은 아이몰 MD가 승인)
* **유의사항**
    * 지정한 약정일로 고객 TMS가 나가니 가능한 날짜로 지정할 것
    * 지정한 약정일까지 미출고 시 익일 새벽 발송 불가
    * 롯데온/백화점몰/아이몰은 2회, 그 외 채널은 1회만 신청 가능
    * 선택 불가한 주문 or 날짜는 담당 MD에게 요청
    * 일반 주문 기준 5영업일 이상 약정일 지정 시 3일 옵션 품절 페널티 부과
* **승인 프로세스 이미지 설명**
    * [백화점 MD 승인/반려] → [백화점 MD 승인/반려] → [백화점 MD 승인/반려] 또는 [아이몰 MD 승인/반려(아이몰 주문 시)]

### 4. 추가 배송 접수 (센터 경유)

* **추가배송이란?**: 본품, 사은품, 쇼핑백 누락 등으로 인해 추가 배송이 필요한 경우를 의미함.
* **추가배송 접수 방법**
    1. PO 주문통합조회 화면에서 추가배송이 필요한 주문 클릭
    2. 주문목록 옆 '추가배송접수' 버튼 클릭 후 팝업창 내용 작성하여 '등록' 버튼 클릭
    3. 추후 집하장에서 라벨 올라오면 추가배송할 상품 기존과 동일하게 집하
* **유의사항**
    * '출고완료' 상태부터 접수가 가능함
    * 접수한 추가배송건들은 PO '추가배송조회' 화면에서 조회 가능
    * AS수선, 심의출고, 심의반송, 부분교환, 맞교환, 배송지 변경은 고객센터로 요청
    * 무분별한 접수 방지를 위해 PO 접수는 주문 순번당 1회만 가능
