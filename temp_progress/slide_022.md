### 1. 오늘 도착 서비스 개요
* **정의:** 매장/업체 준비한 상품을 센터 이동 없이 점포에서 바로 출고하여 더 빠르게 배송하는 서비스

| 구분 | 세부내용 | 비고 |
| :--- | :--- | :--- |
| **서비스내용** | 오전 10시 이전 주문 시, 당일 자정 내 배송 (평일 운영, 수도권 대상, 일부 지역 제외) | |
| **서비스신청방법** | (내용 없음) | |
| **운영프로세스** | 점포에서 상품 포장 후 즉시 출고 및 배송 진행 | |
| **운영점포** | 본점, 잠실, 인천, 노원(나이키) 한정 운영 중 | 집하율 90% 이상 매장 대상 |
| **배송안내** | 오늘 도착 상품은 상품 내 별도 배송 플래그 노출 | 오늘도착 로고 사용 |

### 2. 물류 프로세스 비교

* **센터경유 일반택배**
    * **경로:** [점포] → [센터] → [택배사] → [택배배송] → [고객]
    * **특징:** 매장 집하 시 익일배송 (익일 배송율 97%)
* **오늘 도착**
    * **경로:** [점포] → [당일배송] → [고객]
    * **특징:** 매장 집하 시 당일배송 (당일 배송율 99%)
