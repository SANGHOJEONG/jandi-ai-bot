### [상품등록] 채널상품등록_상품상세설명 순서
1. **상품 카테고리 선택:** 등록할 카테고리를 선택하고 사이즈 입력 후, [미리보기] → [사이즈 조건표 복사] 버튼을 클릭합니다.
2. **사이즈 조건표 데이터 입력:**
   - 상의/하의 사이즈(cm) 항목에 각 브랜드 사이즈에 맞는 데이터를 입력합니다.
3. **사이즈 조건표 미리보기:** 입력된 내용을 확인합니다.
4. **사이즈 조건표 복사:**
   - 입력된 내용 확인 후 [사이즈 조건표 복사] 버튼을 클릭합니다.
   - 클릭 시 자동으로 상품상세설명에 이미지로 입력됩니다.
5. **상품 상세 내용 작성:**
   - 상품 상세는 공통 상품정보 등록 창에서 입력한 이미지 및 텍스트가 자동 복사됩니다.
   - 롯데ON/아이몰에 해당하는 추가 이미지 및 텍스트를 입력합니다.
