## 6. 제휴채널 운영 - 아이몰_상품등록 (점추가)

**기본 정보**
* **대상**: 파트너사
* **작성자**: 신주훈
* **검수자**: 임예린
* **수정일**: 26.06.09

---

### **상품 등록(점추가) 절차**

1. **품번 / 자점 입력하기**
   * 품번란이 1개점 1개 품번인 경우 자동으로 입력됩니다.
2. **상품구분, 상위채널 설정 및 조회**
   * 상품구분: '온라인전용' 선택
   * 상위채널: '파트너오피스' 선택
   * [조회] 버튼 클릭 (화면 우측 상단 5번 위치)
3. **상품 선택 및 점추가 진행**
   * 상품목록에서 점추가 할 상품 선택 (6번 영역)
   * 등록채널 선택: '파트너오피스' 선택 (7번 영역)
   * [점추가] 버튼 클릭 (8번 영역)
4. **정보 입력 및 저장**
   * 이후 아래 공통상품정보 항목에서 [가격 / 마진] 입력
   * [임시저장] 버튼 클릭 (9번 영역)

---

### **화면 구성 안내 (이미지 내 번호 기준)**
* **1번**: 품번 입력란
* **2번**: 자점 입력란
* **3번**: 상품구분 (온라인전용)
* **4번**: 상위채널 (파트너오피스)
* **5번**: [조회] 버튼
* **6번**: 상품목록 영역 (등록된 상품 정보 확인 및 선택)
* **7번**: 등록채널 선택란 (파트너오피스 등)
* **8번**: [점추가] 버튼
* **9번**: [임시저장] 버튼
