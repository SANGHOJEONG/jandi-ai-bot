## 온라인 상품 관리 > 상품 등록(채널상품등록_채널 전시 정보)

- **채널 전시 정보**
  - **전시 상품명**: 고객에게 노출되는 상품명
  - **업체 상품 코드**: 스타일 코드 입력
  - **공동상품제한여부**:
    - 제한
    - 제한 안함
    - ※ '제한'으로 등록 시, 타채널에서 접수가 불가
  - **가격비교 노출여부**:
    - 노출
    - 노출안함
    - ※ 데이터 호스팅/더나아가 다른 비교 사이트에 해당 스타일코드 노출
  - **롯데 ON 전시여부**:
    - 노출
    - 노출안함
    - ※ 롯데 ON 해당 스타일코드 노출
  - **중량**: g 단위 입력
  - **판매시작일**:
    - 판매시작일/시간 입력
    - ※ 등록된 판매 기간에 따라 판매 여부가 결정되는 경우만 해당
  - **최소/최대 구매수량**:
    - 제한없음
    - 제한함
  - **최소구매수량**: 0
  - **판매종료일**:
    - 판매종료일/시간 입력
    - ※ 특정 기간 후 추가 판매를 막는 경우만 해당

- **[상품등록]_채널상품등록_채널 전시 정보**
  - 순서에 따라 진행
  - [가격관리]메뉴에서 예약상품 설정 가능
