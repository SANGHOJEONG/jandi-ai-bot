## 아이몰_상품 등록 (상품 공통정보 등록)

**대상 : 파트너사 | 작성자 : 신주훈 | 검수자 : 임예린 | 수정: 26.06.09**

---

### [왼쪽 영역] 상품공통정보 입력
* **① 스타일코드 입력 후 ② [중복확인] 버튼 클릭**
    * 타점에서 기등록된 스타일코드는 '점추가'만 가능
* **③ 상품명 입력**
* **④ 상품분류 선택**
    * 상품분류 타이핑 후 돋보기 버튼을 클릭하여 선택
    * 돋보기 버튼 클릭 후 전체 상품분류 중 선택
* **⑤ 판매가 입력**
    * 롯데ON/백화점몰의 경우 정책상, 가격변경/세일등록 시 최초가 노출 안 됨
* **⑥ 마진구분 선택**
* **⑦ 마진율 선택**
* **⑧ 연령제한 선택**
    * 청소년 판매불가 상품은 필수 입력
* **⑨ 검색 키워드 선택**
    * 복수 입력 시 ',' 로 분리 (최대 10개 가능)

---

### [오른쪽 영역] 상품정보 및 이미지 등록
* **상품정보제공고시**
    * 소득공제여부, ISBN(서적만 입력), 즐겨찾기 선택, 유형 선택
    * 상품정보제공고시 항목, 설명, 내용 입력
    * '상품상세 참조'로 입력 시 상품승인거부 될 수 있음
    * 전자상거래법에 따라 상품 정보를 소비자에게 제공(표시)하도록 되어 있는 필수 정보
* **상품상세설명**
    * 상품상세내역 입력 / 사진 업로드 시 사진 버튼 클릭 후 JPG 파일로 업로드
    * 브랜드 공홈 또는 타쇼핑몰 이미지 마우스로 드래그 후 복사 붙여넣기 시 링크 접속으로 인한 연동 오류 발생
* **상품이미지**
    * 이미지 규격: 1500x1500픽셀 권장 / JPG 파일로만 등록 가능
    * 다중 이미지 추가 및 일괄 업로드 가능
    * 이미지 순번: 대표 이미지 설정 및 노출순번 변경 가능
    * 업로드 이미지 클릭 시 원본 사이즈로 확인 가능
    * **임시저장:** 모든 상품공통정보 입력 후 [임시저장] 버튼 클릭 (상단 상품목록 위에도 위치)

---

### 주요 절차 요약
1. 스타일코드 입력 후 [중복확인] 버튼 클릭 (타점에서 기등록 된 코드는 '점추가'만 가능)
2. 상품명 입력 / 상품분류 선택 / 판매가 입력 / 마진구분 선택 / 마진율 선택
3. 연령제한 선택 (청소년 판매불가 상품은 필수 입력) / 검색 키워드 선택
