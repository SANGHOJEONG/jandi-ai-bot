## 아이몰_가격관리 (가격/세일/마진)_일괄변경

* **문서 정보**
    * 대상: 파트너사
    * 작성자: 신주훈
    * 검수자: 임예린
    * 수정일: 26.06.09

* **화면 구성 및 주요 기능 (이미지 설명)**
    * **① 품번 / ② 자점 입력하기**: 품번 권한이 1개점 1개 품번인 경우 자동 입력됨.
    * **③ 상위채널 ‘파트너오피스’ 선택 후, ④ 조회 버튼 클릭**: 조회를 통해 상품 목록을 불러옴.

* **업무 매뉴얼: 가격/마진 일괄 변경 방법**

    * **판매가 일괄 변경**
        1. 변경구분에서 ‘판매가 변경’ 선택
        2. 적용시작일 및 마진율 입력 (변경 없을 시 기존 마진율 유지)
        3. 변경 공급가 입력 (할인율 / 정액할인 / 지정가 판매 중 선택)
        4. 변경할 상품 선택
        5. ‘판매가/마진 일괄적용’ 버튼 클릭
        6. ‘저장’ 버튼 클릭

    * **마진율 일괄 변경**
        1. 변경구분에서 ‘마진율 변경’ 선택
        2. 적용시작일 및 변경 마진율 입력
        3. 변경할 상품 선택
        4. ‘판매가/마진 일괄적용’ 버튼 클릭
        5. ‘저장’ 버튼 클릭

* **주의사항**
    * **마진코드/마진율 확인**: 사용 중인 마진코드나 마진율이 보이지 않는 경우, 해당 마진이 종료된 상태이므로 파트리더를 통해 ‘마진 오픈’을 요청해야 함.

* **기타 정보**
