## 아이몰_상품별 재고 관리

* **대상:** 파트너사
* **작성자:** 신주훈
* **검수자:** 임예린
* **수정일:** 26.06.09

---

### 업무 매뉴얼 내용

**1. 기본 조회 방법**
* ① **품번** / ② **점포** 입력하기 (품번권한이 1개점 1개 품번인 경우 자동입력 됨)
* ③ 상위채널 ‘파트너오피스’ 선택 후, ④ **조회** 버튼 클릭

**2. 상품 재고 변경 방법**
* **1개 상품 재고 변경:** ⑤ 재고 입력 후, ⑥ [저장] 버튼 클릭
* **선택 상품 일괄 변경:** ① 재고 입력 후, ③ [선택상품재고 일괄변경] 클릭 후, ⑥ [저장] 버튼 클릭
* **전체 상품 일괄 변경:** ① 재고 입력 후, ④ [선택상품재고 일괄변경] 클릭 후, ⑥ [저장] 버튼 클릭

**3. 유의사항**
* ** 선택 칸이 비활성화 되어 있는 경우:** 발송불가 패널티로 인해 온라인 영업중지가 걸린 상태입니다.

### 화면 캡처 내 정보 (구성요소 설명)
* **상단 검색 필터:** 
    * 품번(입력란), 점포(선택란), 상위채널(파트너오피스 선택), 상품코드구분(전체), 상품상태, 단종상태, 재고수량, 판매시작일 등
* **주요 버튼:**
    * ① 재고수량 변경 (일괄변경용)
    * ③ 선택상품재고 일괄변경
    * ④ 전체상품재고 일괄변경
    * ⑥ 저장
    * 조회(우측 상단 돋보기 아이콘)
* **재고 목록 테이블:**
    * 선택, 상태, 온라인 재고, 스타일코드, 상품명, 옵션, 원가, 데코단품코드, 채널상품코드, 상위채널명, 점포, 품번, 품번명 항목으로 구성
