**개요**
* 대상: 파트너사
* 작성자: 신주훈
* 검수자: 임예린
* 수정일: 26.06.09

---

### 화면 구성 및 기능 설명
화면 상단부의 주요 기능 번호에 따른 설명은 다음과 같습니다.
1. **품번**: 검색할 품번을 입력합니다.
2. **점포**: 해당되는 점포를 선택합니다.
3. **조회**: 설정한 조건으로 데이터를 조회합니다.
4. **도움말**: 매뉴얼 관련 도움말을 확인합니다.
5. **엑셀 다운로드**: 현재 리스트를 엑셀 파일로 내려받습니다.
6. **엑셀 업로드**: 작성된 엑셀 파일을 업로드하여 일괄 등록합니다.
7. **저장**: 변경된 내용을 시스템에 반영합니다.

**기타 입력 정보:**
* **파트너오피스**: 판매 주체를 선택합니다(파트너오피스, CJ ONSTYLE(직), 티몬(직) 등).
* **입력 유의사항**: 
    * 상품코드 입력 시 쉼표(,) 또는 엔터키로 구분합니다 (띄어쓰기 금지).
    * 상품코드 입력 시 특수기호는 사용할 수 없습니다.

---

### 엑셀 파일 일괄 등록 매뉴얼
하단 '엑셀 파일' 표 영역의 각 항목별 입력 규칙입니다.

* **세일/판매가변경 구분**: 
    * 'S': 세일 등록
    * 'C': 판매가 변경
* **정상행사구분**: 
    * '1': 정상
    * '2': 행사
* **판매가/마진**: 
    * 판매가 및 마진 코드를 입력합니다. (마진코드 확인이 필요한 경우 파트너 파트에 문의)
* **적용시작일자 / 적용종료일자**: 
    * 형식: YYYYMMDD (예: 2017012309 -> 2017년 1월 23일 09시)
    * 월/일/시만 입력하며, 세일이 아닌 경우 적용종료일자 입력은 생략 가능합니다.
