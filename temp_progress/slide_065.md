### 2.5 상품등록 및 승인 시 유의사항_생활화학제품

#### 상품공통정보입력 - 생활화학제품
생활화학제품 등록 시 작성해야 할 주요 항목 및 입력 가이드는 다음과 같습니다.

| 상품정보시품목 | 설명 | 내용 입력 가이드 |
| :--- | :--- | :--- |
| **품목 및 제품명** | 품목 및 제품명 모두 입력 | 품명 / 제품명 |
| **용도 및 제형** | 표백제의 경우 계열을 함께 표시 | 용도 / 제형 |
| **제조연월 및 유통기한** | 사용할 경우 유통기한 | 0000 or [별도표기] |
| **중량·용량·매수·크기** | - | - |
| **효과·효능** | 승인대상 생활화학제품에 한함 | 대상 제품인 경우 작성 or [해당없음] |
| **어린이 보호포장 대상 제품 유무** | - | 해당할 경우 작성 |
| **제품에 사용된 화학물질 명칭** | 「안전확인대상생활화학제품 지정 및 안전·표시기준」에 따른 표시대상 화학물질로서 주요 물질, 보존제, 알레르기반응가능물질 등의 명칭 | - |
| **사용상 주의사항** | - | - |
| **안전기준 적합확인 신고번호(자가검사번호) 또는 안전확인대상 생활화학제품 승인번호** | 화학제품안전법 시행일(경과조치 기간 포함) 이전에 생산·수입된 위해우려제품의 경우 종전 법에 따른 자가검사 번호를 표시 | 신고번호 필수 작성 |
| **소비자상담 관련 전화번호** | 담당자 이름 표기가 어려울 경우, 업체명 입력 | 책임자 / 전화번호 |
| **수입자(수입제품에 한함), 제조국 및 제조사** | - | 제조 or 수입국(수입업체) 작성 |

* **필수 확인 사항:**
    * **안전확인대상 생활화학제품 표시**가 없는 경우 등록이 불가합니다.
