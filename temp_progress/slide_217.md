**기본 정보**

*   **작성자:** 신주훈
*   **검수자:** 임예린
*   **수정일:** 26.06.09

**주의 사항**
*   정상 마진 20 / 행사 마진 12 이하 B/D GS(직) 채널은 정책에 따라 입점이 불가합니다.

**업무 프로세스**

*   **1단계: 브랜드(매장) 측으로부터 수취 필요한 정보**
    *   품번 / 매장 전화번호 / 매니저 핸드폰 번호 / 이메일 주소
    *   마진율 / 마진코드 / 매장 파트너포털 아이디
*   **2단계: GS 시스템 내 브랜드 정보 입력**
    *   GS 협력사 정보 - 하위협력사명
    *   GS 협력사 정보 - 담당자 정보
    *   하위협력사 추가 클릭 시 하위협력사 코드 생성
*   **3단계: 상품송신 하위협력사 코드 맵핑**
    *   익일 오전 8시 이후 '매장' 파트너포털 아이디와 맵핑 가능
*   **기타 작업 항목**
    *   **GS 시스템 화면 - 정보변경 - 하위협력사 등록:** 협력사관리 → 정보관리 → 하위 협력사 관리
    *   **GS 시스템 화면 - 하위협력사 추가:** '하위 협력사 관리' 메뉴 이용
    *   **GS 시스템 화면 - 롯데백화점 - 롯데품번관리:** 
        *   하위협력사 생성 후 신규입점 브랜드/점포/하위협력사코드 정보를 GS MD에게 전달
        *   GS 마진율 = 롯데마진율 / 2

**화면 캡처 상세 정보 (하위 협력사 관리)**
*   **경로:** 파트너스 > 협력사 관리 > 정보 관리 > 하위 협력사 관리 탭
*   **주요 기능:** 하위 협력사 정보 리스트 조회 및 '하위협력사 추가' 버튼 지원
*   **기재 항목:** 하위협력사코드, 하위협력사명, 담당자명, 전화번호, 핸드폰번호, 이메일, 팩스번호
*   **기타 사항:** 신규 협력사의 임시 비밀번호는 입력하신 핸드폰 번호로 발송됨
