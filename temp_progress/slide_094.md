### [상품등록] 상품공통정보 입력 가이드
* 모든 입력은 순서에 따라 진행해야 합니다.
* 사용하는 마진율이 보이지 않는 경우, 영업점 파트리더에게 마진키 오픈을 요청하십시오.

### 화면 입력 항목 상세
1. **브랜드스타일코드**: 해당 칸에 코드를 입력합니다.
2. **중복확인**: 등록하려는 상품 스타일코드의 중복 여부를 확인하는 버튼입니다.
3. **상품명**: '상품명_스타일코드' 형식으로 입력합니다.
4. **상품분류**: 돋보기 아이콘을 클릭하여 적절한 상품 분류를 검색 및 선택합니다.
5. **판매가**: 상품의 최초 판매가를 입력합니다.
6. **정상/행사 마진구분**: 해당 항목의 드롭다운 메뉴에서 구분 값을 선택합니다.
7. **마진율**: 설정된 마진구분에 맞는 마진율을 선택합니다.
8. **연령제한**: 청소년보호법에 의해 청소년에게 판매가 제한되는 상품인 경우 반드시 선택해야 합니다.
9. **검색키워드**: 검색어 입력란입니다. (복수 입력 시 ","로 분리하며, 최대 10개까지 가능)
