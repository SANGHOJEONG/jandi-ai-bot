## 파트너포털_ 온라인 상품관리 > 상품등록(채널상품등록_옵션등록)

### [개요]
* **문서 정보:** 대상(파트너사), 작성자(백승현), 검수자(손종현), 수정일(26.06.09)
* **주의사항:** 이 문서는 파트너사 대상 매뉴얼이며, 파트너사 외 외부 유출을 금합니다.

### [진행 순서 및 주의사항]
* 순서에 따라 진행할 것.
* **[직접입력 사용여부]** 반드시 체크할 것.
* **[실제 노출명]**으로 고객에게 노출되므로, 제대로 기입되었는지 최종 확인할 것.

### [화면 상세 설명 및 조작 방법]

#### 1. 컬러칩 정보
* **컬러칩 사용여부:** 사용/사용 안 함 선택
    * *비고:* 컬러칩 등록 브랜드에 한해 '사용' 선택. 컬러칩 사용여부 권한은 담당 MD에게 요청.

#### 2. 옵션 등록 영역
* **주요 버튼:** 추가옵션적용, 행추가, 행삭제, 옵션적용
    * *비고:* 단일 옵션 상품은 '옵션 적용' 후 바로 재고 진행.
* **옵션 입력 테이블:**
    * **옵션유형코드/옵션유형/옵션값:** 색상/색상, 사이즈/사이즈, 직접입력/사이즈/90, 95, 100
    * *주의사항:* 행 추가가 아닌 등록된 기본 색상/사이즈 사용 시, '추가옵션적용' 불가. 옵션 값이 n개인 경우 ','(쉼표)로 분리해서 등록.

#### 3. 롯데 ON 속성정보
* **상단 버튼:** 롯데ON추가옵션변경, 롯데ON추가옵션적용, 롯데ON옵션적용
* **옵션 설정 테이블:**
    * **백화점옵션유형 / 롯데ON옵션유형 / 백화점옵션값 / 롯데ON옵션값 / 직접입력사용여부 / 실제노출명 / 비고**
    * 예시: 사이즈 / 성인 상의 사이즈 / 90 / S / [V]체크 / 90
    * 예시: 사이즈 / 성인 상의 사이즈 / 95 / M / [V]체크 / 95
    * 예시: 사이즈 / 성인 상의 사이즈 / 100 / L / [V]체크 / 100
* **주의사항:**
    * '롯데ON추가옵션적용'을 선택할 경우 [엘롯데/아이몰]옵션 변경이 요청됨.
    * 위에 등록한 옵션 유형/값과 동일한 롯데ON 옵션 유형/값 등록 필수.
    * 각 옵션 유형에 맞는 값으로 선택 필수.
