## 2. 물류/배송: 집송

**문서 정보**
*   **대상:** 파트너사
*   **작성자:** 강민호
*   **검수자:** 강민호
*   **수정일:** 26.06.10

---

### 1. 개요
배송 유형은 전부 '센터경유'이나, 거래처 출고 형태는 하위 3가지 기준으로 나뉩니다.

| 출고형태 | 내용 | 비고 |
| :--- | :--- | :--- |
| **센터경유** | 매장 주문상품 피킹, 점포 집하장 상품 전달/출고 | |
| **집송** | 업체 물류센터 주문상품 피킹, 백화점 셔틀차량 상품 픽업/출고 | |
| **직입고** | 업체 물류센터 주문상품 피킹, 업체 차량 사용-백화점 물류센터 입고/출고 | |

---

### 2. 집송/직입고 업체 참고사항
*   **신청 절차:** 온라인주문 영업 담당자, 집송 가능여부 물류 파트 문의 → 운행 루트 및 적재 환경 검토(물류 파트) → 승인/반려
*   **집송 반려 시, 직입고 대응 가능:** 물류센터 내부 운영 프로세스 집송·직입고 동일. 업체 차량 활용, 백화점물류센터 입고 가능 (17:00 이전)
*   **주문라벨 부자재 공급:** 집송/직입고 물류센터 요청 시 주문라벨 부자재 별도 공급 (박스는 제공하지 않음)
*   **집송 운영 물량:** 최소 월 100건 이상 물량 수행 조건 (운영 중 월 100건 미만 발생 시 운행이 종료될 수 있음)

---

### 3. 집송 프로세스

#### 출고
1. **(업체 물류센터)** 집하예정리스트/주문라벨 출력
2. **(업체 물류센터)** 상품 피킹 - 주문라벨 부착
3. **(업체 물류센터)** 집하검수(집송)
4. **(백화점 물류센터)** 상품 박스 포장 및 출고
5. **고객 택배배송**

#### 반품
1. **고객 신청 반품** 택배회수
2. **(백화점 물류센터)** 반품 수령 및 업체별 분류
3. **(업체 물류센터)** 반품 인수
4. **(업체 물류센터)** 반품 검수 / 확정
5. **고객 환불**
