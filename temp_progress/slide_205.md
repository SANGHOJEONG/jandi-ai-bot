* 작성자: 임예린
* 검수자: 신주훈
* 수정일: 26.06.09

---

### 1. CJ(직) 상품 등록 - 신규등록

* **등록 절차**
    1. 파트너포털 상품 등록 메뉴로 이동합니다.
    2. 상품공통정보등록 화면에서 '상품구분'을 **온라인 전용**으로 선택합니다.
    3. '상위채널'을 **CJ**로 선택하고, '등록채널'에서 **CJ**를 선택합니다.
    4. **신규** 버튼을 클릭하여 생성합니다.
    5. 하단 공통 상품 정보 전체를 입력한 후 **임시저장**을 클릭합니다.
    6. 이후 CJ 창으로 들어가 CJ 전용 상품 등록을 진행합니다.
* **주의사항**
    * 품번/자점 확인 불가 시 브랜드 본사 측 품번 권한 등록이 필요합니다.

---

### 2. 1.1 CJ(직) 상품 등록 - 채널추가 / 점추가

* **등록 절차**
    1. 파트너포털 상품 등록 메뉴로 이동합니다.
    2. 상품공통정보등록 화면에서 '상품구분'을 **온라인 전용**으로 선택합니다.
    3. '상위채널'을 **CJ**로 선택하고, '등록채널'에서 **CJ**를 선택합니다.
    4. **점추가 / 채널추가** 버튼을 클릭하여 생성합니다.
    5. 공통상품정보 가격/마진 입력 후 **임시저장**을 클릭합니다.
    6. 이후 CJ 창으로 들어가 CJ 전용 상품 등록을 진행합니다.
* **주의사항**
    * **점추가:** 타 채널에서 판매 중인 상품만 점추가가 가능합니다.
    * **채널추가:** 당점에서 판매한 타 채널 상품만 추가가 가능합니다.
