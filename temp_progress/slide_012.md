## 온라인 상품 관리 > 상품 등록 (상품 공통 정보)

*   **대상 : 파트너사**
*   **[상품등록] 상품 공통 정보**
    *   순서에 따라 진행
    *   서적 상품 등록 시, 'ISBN' 입력
    *   상품정보고시 내용에 '상품상세참조'로만 기입 시, 상품 승인 거부 가능
    *   모든 상품공통정보 입력 후 [임시저장] 버튼 클릭

### 상품정보고시등록

*   **상품정보고시 등록명**
*   **설명**
*   **유형 선택**
*   **내용:** 선택 유형에 맞는 상품정보고시 내용 모두 입력
    *   **(이미지)** [사진] 버튼 클릭 후 JPG 파일로 업로드

### 상품상세설명

*   (WYSIWYG 편집기)
*   상품상세내역 입력 / 사진 업로드 시

### 상품이미지

*   **+다중이미지추가**
*   **이미지등록가이드**
*   **이미지순번 : 대표**
    *   (이미지 미리보기 영역)
    *   1,500x1,500 픽셀, 1MB 이하 권장 (미리보기)
    *   (이미지 추가/삭제 버튼 및 픽셀/크기 옵션)
