### 1. CJ(직) 상품 관리 – 상품별 재고 관리

*   **진행 순서**
    1. 영업통합 시스템 접속
    2. [상품별 재고 관리] 메뉴 이동
    3. 품번, 점포, 상위채널(CJ ONSTYLE(직)) 세팅 후 조회
    4. 선택 체크 후 재고 입력 -> [선택상품 재고 일괄 변경] 버튼 클릭
*   **주의사항**
    *   선택 칸이 비활성화된 경우: 채널 MD에게 잠금해제 요청 (주로 품절인 경우)
    *   우측 돋보기 아이콘 클릭 시: 재고 이력 확인 가능

---

### 2. CJ(직) 상품 관리 – 상품상태 관리

*   **진행 순서**
    1. 영업통합 시스템 접속
    2. [GS/CJ/티몬 상품관리] 메뉴 이동
    3. 품번, 점포, 상위채널(CJ ONSTYLE(직)) 세팅 후 조회
    4. 상품상태 변경 (판매중, 전체품절, 판매중단 등 선택)
*   **참고 사항**
    *   GS SHOP의 경우 단품의 재고가 0인 경우 상태 변경이 불가하므로, 재고 등록 후 변경 필요
*   **주의사항**
    *   선택 칸이 비활성화된 경우: 채널 MD에게 잠금해제 요청 (주로 품절인 경우)
    *   우측 돋보기 아이콘 클릭 시: 재고 이력 확인 가능
