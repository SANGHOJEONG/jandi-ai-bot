## 6. 제휴채널 운영 - GS(직) 상품 등록

**[기본 정보]**
*   **대상:** 파트너사
*   **작성자:** 신주훈
*   **검수자:** 임예린
*   **수정일:** 26.06.09

---

### 1. GS(직) 상품 등록 - 신규등록

*   **파트너포털 상품 등록 절차 (상품공통정보등록):**
    1.  상품구분에서 '온라인 전용'을 선택합니다.
    2.  상위채널에서 'GS'를 선택합니다.
    3.  '신규' 버튼을 클릭하여 생성합니다.
    4.  하단 공통 상품 정보 전체를 입력한 후 '임시저장'을 클릭합니다.
    5.  GS 창으로 들어가 GS 전용 상품 등록을 진행합니다.
*   **주의사항:** 
    *   품번/자점 확인 불가 시 브랜드 본사 측 품번 권한 등록이 필요합니다.

---

### 2. GS(직) 상품 등록 – 채널추가 / 점추가

*   **파트너포털 상품 등록 절차 (상품공통정보등록):**
    1.  상품구분에서 '온라인 전용'을 선택합니다.
    2.  상위채널에서 'GS'를 선택합니다.
    3.  '점추가 / 채널추가' 버튼을 클릭하여 생성합니다.
    4.  공통상품정보(가격/마진) 입력 후 '임시저장'을 클릭합니다.
    5.  GS 창으로 들어가 GS 전용 상품 등록을 진행합니다.
*   **주요 용어 설명:**
    *   **점추가:** 타 채널에서 판매 중인 상품만 점추가 가능합니다.
    *   **채널추가:** 당점에서 판매한 타 채널 상품만 추가 가능합니다.
