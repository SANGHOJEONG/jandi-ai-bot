### 거래처 사용자 등록 상세 내용

**1. 거래처 정보**
* 거래처유형: 상위거래처
* 거래처구분: 하위거래처
* 상위거래처명: 롯데쇼핑(주)본점 (10002)
* 하위거래처명: 본점(식품_부경드림) (56374)

**2. 사용자 정보 입력 항목**
* **사용자 ID (1):** 사용자 ID 입력 후 **중복확인 (2)** 버튼 클릭 필수
* **사용자명 (3):** 사용자명 입력
* **비밀번호:** 비밀번호 설정
* **직책 / 부서 / 호칭:** 관련 정보 입력
* **이메일 / 전화번호 / 내선번호:** 연락처 정보 입력
* **휴대폰번호 (4):** 휴대폰 번호 입력
* **담당업무:** 담당 업무 선택 (예: 매장/주문 담당자)
* **SMS 수신여부:** 수신 또는 수신안함 선택
* **입사일자:** 입사일 선택
* **퇴사일자:** 퇴사 시 입력
* **근무상태:** 재직 여부 선택
* **사용자구분:** 일반사용자

**3. 하단 버튼**
* **취소:** 등록 취소
* **저장 (5):** 입력한 사용자 정보 저장
