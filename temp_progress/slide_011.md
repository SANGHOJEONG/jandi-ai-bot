## 6.1 파트너 포털 - 온라인 상품 관리 > 상품 등록 (상품공통정보)

*   **[상품등록]_상품공통정보**
    *   순서에 따라 진행
    *   사용하는 마진율이 보이지 않는 경우, 영업점 파트너에게 마진키 오픈 요청
*   **상품공통정보등록 탭**
    *   **브랜드명:** 리복
    *   **상품명:**
        *   상품명_스타일코드
        *   스타일코드 입력
        *   중복확인 버튼
    *   **상품분류:**
        *   상품분류 입력
        *   돋보기 아이콘 (검색)
    *   **판매가:**
        *   판매가 입력
        *   상품의 최초 판매가를 입력해주세요.
    *   **정상/할사 마진구분:**
        *   마진구분 선택 (드롭다운)
        *   마진율 선택 (드롭다운)
*   **온라인 상품 기본 정보**
    *   **연령제한:**
        *   연령제한 선택 (드롭다운)
        *   청소년 보호법에 의해 청소년에게 판매가 제한되는 상품 등록 시 꼭 선택해주세요.
    *   **검색키워드:**
        *   검색어 입력 (복수 입력 시 "," 로 분리 / 최대 10개 가능)
