제공해주신 슬라이드의 내용을 마크다운 형식으로 정리해 드립니다.

## 4. 온라인 신규 입점: 온라인 입점 프로세스 (파트너사)_ 3단계

### 3. 세이프빌(Safebill) 가입
* **세이프빌 사이트:** [www.safebill.co.kr](http://www.safebill.co.kr)
* **전자세금계산서 가입 참고:** 아이디는 'dt+사업자번호'로 생성 (예시: dt1048126067)

#### 상세 가입 절차
1.  **Safebill 접속**
    * 사이트(www.safebill.co.kr) 접속 후 로그인 정보 아래의 '회원가입'을 클릭합니다.
2.  **사업자번호 조회**
    * 사업자 번호를 입력하고 조회를 클릭합니다.
3.  **사용자 등록 신청**
    * '신청하기'를 클릭합니다.
4.  **약관동의 절차**
    * 약관을 읽어보고 난 후, '동의합니다'란에 체크합니다.
5.  **회원정보 입력 - 아이디 생성**
    * 아이디와 비밀번호를 입력합니다.
    * 아이디 생성 시 'dt+사업자번호'로 생성합니다.
    * **주의:** 이미 아이디가 있는 경우에는 추가 생성을 클릭(22페이지 참조)합니다.
6.  **회원정보 입력 - 기업 관리자 정보 입력**
    * ① 담당자 정보 입력: 세금계산서 담당자의 휴대폰 번호와 이메일 주소를 정확하게 입력해야 합니다.
    * ② 공인인증서 등록여부: '나중에 제출합니다.'에 체크합니다.
    * ③ 신청 버튼: 버튼을 클릭하면 가입이 완료됩니다.
