## 파트너포탈_ 온라인 매출 관리 > 주문조회 (온라인)

**[문서 정보]**
*   **대상:** 파트너사
*   **작성자:** 백승현
*   **검수자:** 손종현
*   **수정일:** 26.06.09

### 1. 화면 구성 설명 (주문조회 페이지)
화면 상단 검색 필터를 통해 주문 정보를 조회할 수 있습니다.
*   **① 파트너오피스:** 파트너사 선택 항목
*   **② 점포:** 점포 선택 및 검색
*   **③ 품번:** 품번 입력란
*   **④ 조회 버튼:** 검색 조건 설정 후 클릭하여 데이터 조회

**[참고 사항]**
*   주문일자, 상품코드 등 상세 검색 조건 활용 가능
*   상품 코드는 쉼표(,) 혹은 엔터키로 구분하여 여러 개 입력 가능 (예: 95, 100, 105)
*   특수기호 입력 불가 (입력 시 별도 연락 필요)

### 2. 주문건수 요약
주문 상태별 현황을 한눈에 확인할 수 있습니다.
*   **주문완료 / 출고지시 / 상품준비 / 발송약정 / 발송완료 / 고객인도 / 예약주문 / 클레임 주문현황 / 발송대기 / 주문취소 / 발송불가 / 교환/반품 / 전체** 등 단계별 수치 표시

### 3. 주문 목록 상세
조회된 주문 리스트는 다음 항목들을 포함합니다.
*   채널주문번호, 점포, 품번, 스타일코드, 하위채널, 상품명, 주문일시, 주문구분, 주문상태, 주문상태변경일시, 출고지시일, 옵션

### 4. [주문조회(온라인)] 핵심 가이드
*   **진행 순서:** 시스템에 표시된 순서에 따라 업무를 진행합니다.
*   **주문 확인:** '주문' 단계부터 확인이 가능합니다.
*   **상품 준비:** 주문 확인 후 상품 준비를 진행합니다.
*   **주의 사항:** **출고지시가 떨어지지 않은 주문은 절대 출고 금지**합니다.
