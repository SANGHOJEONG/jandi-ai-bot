## 6. 제휴채널 운영 - 아이몰_채널 상품 등록

*   **대상:** 파트너사
*   **작성자:** 신주훈
*   **검수자:** 임예린
*   **수정일:** 26.06.09

---

### 1. 채널 전시 정보
*   **① 전시 상품명:** 고객에게 노출되는 상품명을 입력합니다.
*   **② 업체 상품 코드:** 해당 상품의 스타일코드를 입력합니다.
*   **③ 공동상품제한여부:** '제한' 선택 시 타 몰에서 이미 접수가 진행 중인 상품은 등록이 불가합니다.
*   **④ 가격비교 노출여부:** 네이버 쇼핑/다나와 등 가격비교 사이트에 해당 스타일코드 노출 여부를 설정합니다.
*   **⑤ 전시매장 노출여부:** 롯데백화점몰 해당 스타일코드 노출 여부를 설정합니다.
*   **⑥ 롯데ON 전시여부:** 롯데ON 해당 스타일코드 노출 여부를 설정하며, '가격비교 노출여부/전시매장 노출여부/롯데ON전시여부'는 '롯데ON전시관리' 메뉴에서 수정 가능합니다.
*   **⑦ 판매시작일/시간:** 특정일 이후 판매가 진행되어야 하는 경우 설정하며, 상세한 설정은 [가격관리] 메뉴에서 가능합니다.
*   **⑧ 판매종료일/시간:** 특정 기간 후 추가 판매를 안 하는 상품의 경우 설정합니다.
*   **⑨ 최소/최대 구매수량:** 제한 여부를 설정합니다.
*   **⑩ 최소구매수량:** 최소 구매수량을 입력합니다.
*   **⑪ 최대구매수량:** 최대 구매수량을 입력합니다.

---

### 2. 상품상세설명
*   **상품상세 이미지에 사이즈 조절표 이미지를 입력하는 경우:**
    *   ① 등록할 카테고리를 선택하고 사이즈 입력 후, [미리보기] → [사이즈 조절표 복사] 버튼을 누릅니다.
    *   ② **사이즈 조절표 데이터 입력:** 신상/신사경(상의/하의) 사이즈(cm)를 입력합니다.
    *   ③ **사이즈조절표 미리보기:** 입력된 내용을 확인합니다. (웹 소스 방식이나 텍스트 타입으로 조정 가능)
    *   ④ **사이즈조절표 복사:** 입력된 내용을 확인 후 해당 버튼을 클릭하면 상품상세에 자동으로 입력됩니다.
    *   ⑤ **이미지 등록:**
        *   상품 상세는 공통상품정보등록 창에서 입력한 이미지 및 텍스트가 자동 복사됩니다.
        *   롯데ON/아이몰에 해당하는 추가 이미지 및 텍스트를 입력합니다.
        *   이미지 추가 시에는 동일하게 오른쪽 상단 [사진] 버튼 클릭 후 JPG 파일로 업로드합니다.
