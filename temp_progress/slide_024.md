### 1. 매장 픽업 개요
*   **정의:** 온라인몰에서 주문한 상품을 매장에서 픽업하는 서비스

#### 주문 처리 프로세스
| 주문상태 | 처리주체 | 변경시점 |
| :--- | :--- | :--- |
| **출고지시** | 고객 | 고객 결제완료 |
| **상품준비중** | 매장 | PO 스마트픽매장 발송처리화면에서 ‘상품준비 버튼클릭’ |
| **출고완료** | 매장 | PO 스마트픽매장 발송처리 화면에서 ‘스토어픽사용/반품처리’ 버튼 클릭 후 교환권 번호 입력 후 고객인도처리 |
| **배송완료** | 자동처리 | 출고완료 시 자동 처리 |
| **회수지시~회수확정** | - | 일반 주문 프로세스와 동일 |

---

### 2. 업무 유의사항 및 지침
1. **신속한 상품 준비:** 픽업 가능한 주문은 최대한 빨리 상품준비 처리를 해야 하며, 당일 픽업 주문을 18시까지 미처리 시 주문이 자동 취소됩니다.
2. **TMS 발송:** 주문 상품준비 미처리 TMS는 매일 09, 10, 12, 14, 16, 17시에 발송됩니다. 미수신 시 PO 계정 연락처를 최신화해야 합니다.
3. **미수령 주문 처리:** 픽업 예정일이 경과된 주문은 2일차 새벽에 자동 취소됩니다. 예정일 익일 고객에게 미수령 TMS가 발송됩니다.
4. **문의처:** 문의 및 요청사항은 스마트픽 고객센터(1599-8437)로 연락 바랍니다. (운영시간: 평일 10~19시 / 주말 10~20시 ※ 점심시간 12~13시)
5. **상품 등록 권한:** 매장픽업 상품 등록은 파트너 포털에서 가능하며, 권한은 담당 MD에게 요청해야 합니다.
6. **특판 주문:** 특판은 반드시 매장픽업으로만 처리해야 합니다(아울렛, 업체배송 제외). 일반주문으로 생성 시 배송완료 처리가 불가합니다.
7. **기타 픽업 유형:** 내주변픽업, 리버스픽업은 매장에서의 일반 주문 출고, 반품과 동일한 프로세스이며, 픽업/반품지가 세븐일레븐(편의점)이라는 점만 차이가 있습니다.
