## 6.1 파트너 포털

### 온라인 매출 관리 > 매출조회(온라인)

#### 화면 캡처 내용

*   **LOTTE**
*   **메뉴:** 기초정보, 온라인 채널 연계, 전선적배달, 발주현대, 납품/배송관리, 대금/세금 회계, 파트너 네트웍, 사전관리, 매출세무 가이드
*   **현재 페이지:** 홈 > 온라인몰 연계 > 주문/배송관리 > 매출조회(온라인)
*   **검색 조건:**
    *   **채널:**
    *   **주문 구분:**
    *   **주문 일자:** 2020-04-07 ~ 2020-04-14
    *   **매출 일자:** 2020-04-07 ~ 2020-04-14
    *   **파트너 명:** 텍스트 입력 창
    *   **브랜드:** 텍스트 입력 창
    *   **상품 그룹:** 텍스트 입력 창
    *   **상품 선택:** 텍스트 입력 창
*   **전체/부분 선택:** 전체, 부분
*   **날짜 필터:** 당일, 1주일, 1개월, 3개월
*   **배송 완료:** 1000 (텍스트 입력 창)
    *   "※ 상품별 코드 입력 (RAOK.) 혹은 택대기구로 구매해서
        피어스기 건이 입력하세요.(예. 95,100,105)
        상품별 코드가 없다면 주문 번호로 입력하시면 됩니다.
        등록 불가한 아이템은 (RAOK.) 부분은 비워두기"
*   **매출 현황:**
    *   배송 현장: 12 건수, 3,725,670 금액
    *   반품 현장: 2 건수, -514,800 금액
*   **매출 목록:**
    *   **번호, 주문 구분, 상세내역 번호, 배송일자, 하위 채널, 상품코드, 스타일, 금액, 구분, 정산가, 오류 구분, S+위, 제공, 적용, (배송마감일)**
    *   (데이터 표)

#### 팝업 설명

*   **[매출조회(온라인)]**
    *   순서에 따라 진행
    *   파트너오피스 주문 건에 대한 파트너포탈 매출 편칭
    *   → '배송완료' 완료 시점
    *   미등록 매출 발생 시, 파트너리더에게 매출 수정 요청
